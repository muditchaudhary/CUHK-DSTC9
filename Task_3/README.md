# Task 3 instructions

### To train the model:
./train_multiknow.sh

### To run the model:
run ./run_multiknow.sh
and you will get the GPT2-XL output

### To get GPT2-XL-RR and Ens-GPT:
RR: run ./replace.sh with --mode RR
Ens-GPT: run ./replace.sh with --mode Ens

### To score the output:
run ./score.sh
